PROJECT CASE STUDY

Portal CRQ - Simplifica AI: Engineering Autonomy Hub | Built at Itaú Unibanco

Tech Stack:Java 21Virtual ThreadsSpring Boot 3.xReactTailwind CSSDocusaurusJUnitPitestTestcontainersLocalStackTerraformAWS ECS FargateAWS API GatewayS3GitHub Actions

Business Problem

The Lending division suffered from scattered operational utilities, slow manual documentation, and barriers to checking architectural health or incident logs. Squashing these pain points demanded a unified portal that centralized test data configurations, runbooks, compliance checkups, and guide books under a self-service design.

Technical Solution

Developed a Solutions Hub under Hexagonal Architecture on the backend (Java 21, Spring Boot, and Virtual Threads on ECS Fargate) linked to a static React web portal hosted on GitHub Pages that securely interfaces via AWS API Gateway proxies. The backend encapsulates: 'massa' (parallelized data generator), 'runbook' (S3 persistence of JSON War Room files via POST/GET/DELETE), 'agentchatbot' (contextual LLM integrations with custom personas), and 'arquitetura' (complex evaluator scanning organization repositories via GitHub REST API, scoring 6-month production releases against Lending reference architectures, tracking codeowners, SDD, and BMAD compliance, and writing JSON scorecards to S3). The frontend Lighthouse tab embeds Amazon QuickSight reports alongside a 9-block incident workbook.

Impact Metrics

Achieved a unified and modular architectural hub. Distributed reference blueprints using a custom CLI tool. Automated architectural diagnostics for 100% of Lending repositories. Reached 95% unit test coverage (JUnit) and 90% mutation coverage (Pitest) across backend domain services.
